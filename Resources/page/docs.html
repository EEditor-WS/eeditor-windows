Документация по EEditor

Начало работы

EEditor - это крутейший альтернативный неофициальный редактор сценариев для Warnament. В отличие от встроенного в игру редактора, EEditor предоставляет множество уникальных возможностей:

  • Мощные инструменты для редактирования стран и их отношений
  • Расширенные настройки реформ и их эффектов
  • Множество кнопок, ускоряющих создание сценариев
  • Удобный пользовательский интерфейс
  • Поддержка мультиязычности
  • И многое другое, чего нет в оригинальном редакторе

Всё необходимое будет добавлено позже, а пока ознакомьтесь с этим текстом.

Установка

Для начала работы с EEditor вам потребуется:

  • Современный веб-браузер (Chrome, Firefox, Edge)

Требования

Минимальные требования:

  • Веб-браузер с поддержкой HTML5
  • JavaScript включен
  • Стабильное интернет-соединение

Первые шаги

Чтобы начать работу:

  1. Откройте EEditor в браузере
  2. Создайте новый сценарий или загрузите существующий
  3. Ознакомьтесь с интерфейсом редактора

Консольные команды

Базовые команды

Основные команды для управления игрой:

  • /commands - Показать список всех доступных команд
  • /m name_or_id - Отправить личное сообщение игроку
  • /reset_timer - Сбросить таймер игры

Читы и отладка

Команды для тестирования и отладки (не работают в мультиплеере):

  • /resource - Получить 9999 всех ресурсов
  • /cheat - Открыть все технологии и получить безлимитные значения
  • /debug_ai - Показать расширенную информацию о ботах в меню провинции
  • /debug_land_names - Показать прямоугольники для отрисовки
  • /draw_borders - Показать/скрыть границы
  • /hide_interface - Скрыть интерфейс на 5 секунд (для скриншотов)
  • /change_country - Сменить страну (выберите провинцию нужной страны)
  • /lost - Показать анимацию поражения
  • /spawn <army_type> <count> - Создать армию в выбранной провинции
    • Типы армий: infantry, tank, cruiser, battleship, aircraft_carrier
    • Пример: /spawn battleship 3
  • /results_screen - Показать экран результатов как в мультиплеере
  • /nuclear - Показать анимацию ядерного оружия (без урона)
  • /nuclear_land - Показать ядерную анимацию для всех провинций выбранной страны
  • /downed_plane - Показать анимацию сбитого самолета
  • /fog_of_war - Включить/выключить отображение армии и построек других стран

Мультиплеер

Команды для управления многопользовательской игрой:

  • /info [JM+] - Получить информацию об игроке
  • /history [JM+] - Получить историю игрока
  • /kick name_or_id [JM+] - Выгнать игрока
  • /ban name_or_id time reason [JM+] - Забанить игрока
  • /banid id hours reason [JM+] - Забанить игрока по ID
  • /banip name_or_id hours reason [M+] - Забанить игрока по IP
  • /unban id [M+] - Разбанить игрока
  • /mute name_or_id [M+] - Заглушить игрока
  • /unmute name_or_id [M+] - Снять заглушение с игрока

Админские команды

Команды для администраторов:

  • /content - Показать список доступного контента
  • /setdifficulty <difficulty> - Установить сложность игры
    • Доступные значения: easy, standard, hard, impossible
  • /enable_action - Включить определенные действия в игре
  • /disable_action - Выключить определенные действия в игре
  • /over - Завершить текущую игру
  • /new_game id_of_scenario - Начать новую игру с указанным сценарием
  • /delaynext - Отложить следующий ход
  • /forcenext - Принудительно начать следующий ход
  • /set_country name_or_id - Изменить страну игрока
  • /a message - Отправить сообщение администратору

Роли админов

Описание ролей и их возможностей:

  • [L] - Лидер
    • Выдача ролей другим игрокам
    • Все возможности нижестоящих ролей
  • [A] - Администратор
    • Управление игровым процессом
    • Настройка параметров игры
    • Все возможности нижестоящих ролей
  • [M] - Модератор
    • Управление банами по IP
    • Управление мутами
    • Все возможности младшего модератора
  • [JM] - Младший модератор
    • Базовые команды модерации
    • Просмотр информации об игроках
    • Управление киками и банами

Сценарии и моды

Команды для работы со сценариями и модификациями:

  • /export_save - Экспортировать сохранение в scenario.save
  • /import save - Импортировать сохранение из scenario.save
  • /patch_scenario <scenario_name.json> - Адаптировать сценарий к новой версии карты
  • /cinema <scale_change> <duration> <x> <y> - Показать анимацию движения карты
    • Пример: /cinema 2 3 -400 500
  • /check_event_req <id> - Проверить требования для события
  • /show_event <id> - Показать событие

Редактор карт

Обзор

Редактор карт — это отдельное приложение для создания игровых карт в формате .map для Warnament. Он позволяет создавать и редактировать провинции, настраивать их параметры и тестировать готовые карты.

Главное меню редактора карт

Создание карты

Процесс создания новой карты состоит из нескольких этапов:

Интерфейс редактирования карты

Подготовка файлов

  1. Откройте Редактор карт
  2. В нижней части окна вы увидите путь к пользовательским файлам
  3. Поместите необходимые файлы в этот путь:
  • Для создания карты из изображения: поместите файл с названием source.png
  • Для редактирования существующей карты: поместите файл output.map

Для редактирования существующей карты:

  • Зайдите в "Map Creator"
  • Нажмите "Импорт"

Работа с провинциями

Создание первой провинции
Добавление второй провинции

Создание провинции

  1. Нажмите "New province"
  2. Нажмите "Add vertice"
  3. Создайте контур провинции, расставляя вершины
  4. Убедитесь, что центр провинции (фиолетовый квадрат) находится внутри её границ

Работа с анклавами

Анклав — это часть провинции, отделённая от её основной территории.

Создание основной части провинции
Добавление первого анклава
Добавление второго анклава
  1. Создайте основную часть провинции
  2. Нажмите кнопку ">>" под текстом "Текущий анклав"
  3. Выберите enclave_2 в списке
  4. Создайте контур анклава
  5. Повторите процесс для каждого дополнительного анклава

Водные провинции

Водные провинции создаются так же, как и обычные, но имеют некоторые особенности:

Создание водной провинции
Добавление острова в водную провинцию
  • Должны соприкасаться с другими провинциями через общие вершины
  • Могут содержать острова (как анклавы)
  • Рекомендуется использовать "High draw priority" для корректного отображения

Управление

Работа с провинциями

  • << и >> - Переключение между анклавами
  • New province - Создание новой провинции
  • Remove Province - Удаление провинции
  • Select Province - Выбор провинции
  • High draw priority - Приоритет отрисовки (для островов/озёр)

Работа с вершинами

  • Add vertice - Режим добавления вершин
  • Move vertice - Перемещение выбранной вершины
  • Remove Vertice - Удаление вершины
  • Select vertice - Выбор вершины
  • Set province center - Установка центра провинции
  • Autofinish border - Автоматическое завершение границы
    • При добавлении вершины автоматически добавляет путевые точки, если новая точка и две предыдущие находятся в одном анклаве
    • При зажатом Shift: добавьте две общие вершины границы между провинциями, затем с зажатым Shift нажмите на последнюю общую точку - редактор автоматически проставит все промежуточные точки

Горячие клавиши

  • Shift + ЛКМ - Автозавершение границ
  • H - Скрыть/показать вершины
  • J - Скрыть/показать полигоны

Верхняя панель

  • Find province - Поиск провинции
  • Map - Страница "Map"
  • Provinces - Страница "Provinces"
  • Draw - Страница "Draw"
  • Transform - Страница "Transform"
  • Additional - Страница "Additional"

Страница "Map"

  • Import - Импорт файла output.map
  • Export - Экспорт в output.map
  • Load backup - Загрузить резервную копию
  • Map name - Название карты в игре
  • Size multiplier - Множитель размера иконок (0.1-3)
  • Show/hide game objects - Показать/скрыть вершины
  • Show/hide province centers - Показать/скрыть центры провинций

Страница "Draw"

  • Create province - Создать провинцию
  • Smooth - Сгладить границу (уменьшить количество вершин). Нужно обвести область или провинцию, чтобы сгладить её границу
  • Simplify - Упростить границу (уменьшить количество вершин). Нужно обвести область или провинцию, чтобы упростить её границу
  • Repiar border - Исправить границу

Страница "Transform"

  • Reload source.png - Перезагрузить фоновое изображение
  • Resize map: x - Изменить ширину карты
  • Resize map: y - Изменить высоту карты
  • Scale the image too - Изменить масштаб изображения, при изменении масштаба карты
  • Resize the image too - Изменить размер карты, согласно введённым выше значениям
  • Move verticls: x - Переместить вершины по оси X
  • Move verticls: y - Переместить вершины по оси Y
  • Move - Переместить вершины, согласно введённым выше значениям

Страница "Additional"

  • Optimization - Можно отключить части карты, которые не видны на фоновом изображении

Тестирование карты

Существует два способа протестировать созданную карту:

Просмотр карты в редакторе

В редакторе карт

  1. Переименуйте файл в output.map
  2. Откройте режим просмотра карт (Map Viewer)

В игре

  1. Переместите файл карты в соответствующую папку игры
  2. Запустите игру
  3. Выберите карту в меню "Новая игра"
